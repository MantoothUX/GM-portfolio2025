# GM-portfolio2025

Greta Mantooth portfolio website

## About

A modern portfolio website built with React, TypeScript, and Vite. Features a dynamic home page, work showcase, and individual project detail pages.

## Tech Stack

- React 19
- TypeScript
- Vite
- React Router
- Framer Motion
- Tailwind CSS

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Preview

```bash
npm run preview
```

## Project Structure

- `src/components/` - React components
- `src/pages/` - Page components (Home, Work, Project)
- `src/data/` - Project data and content
- `archive/` - Archived code and templates
